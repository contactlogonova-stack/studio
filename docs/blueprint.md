# **App Name**: Design Edge Academy

## Core Features:

- Landing Page: Showcase training details (theme, organizers, schedule, location, costs, contacts, social media links).
- Countdown Timer: Implement a countdown timer to the start date (January 17, 2026) to create urgency.
- Registration Form: Collect participant information (Name, First Name, WhatsApp number).
- Payment Modal: Display QR code placeholders (TMoney and Moov) with payment instructions. Participants must pay 2000 F CFA.
- WhatsApp Redirection: Redirect to WhatsApp with pre-filled message including name and confirmation details.
- FormSubmit Integration: Send registration information to the specified email (contact.logonova@gmail.com) using Formsubmit.
- Confirmation Message: Display a thank you message and a response time expectation.

## Style Guidelines:

- Primary color: Deep violet (#673AB7) to convey a premium and modern feel, relating to creativity and digital space.
- Background color: Light violet (#EDE7F6) for a soft and clean backdrop, complementing the primary color.
- Accent color: Electric blue (#3D5AFE) to highlight key actions and interactive elements, creating visual interest.
- Body and headline font: 'Poppins' (sans-serif) for a geometric and modern design, ensures readability and a contemporary style.
- Use clean, minimalist icons that reflect design elements and digital interfaces. Consistent style across all icons.
- Employ a clean, mobile-first, responsive design to provide a premium feel, with well-spaced elements for better readability.
- Incorporate smooth transitions for modals and buttons to guide the user, creating a seamless experience.
